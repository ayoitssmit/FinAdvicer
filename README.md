# FinAdviser

FinAdviser is a financial advice application designed to help users manage their finances, view dashboards, and project future financial states.

## Features

- **Dashboard**: Comprehensive view of financial status.
- **Projections**: Tools to project future financial scenarios.
- **Authentication**: Secure Login and Signup functionality.
- **Responsive Design**: Detailed Navbar and Footer for non-app views.

## Tech Stack

- **Frontend**: React, Vite
- **Routing**: React Router DOM
- **Charts**: Chart.js, React Chartjs 2

## Getting Started

### Prerequisites

- Node.js installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ayoitssmit/FinAdvicer.git
   ```
2. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173` (default Vite port).

## Project Structure

- `frontend/`: Contains the React application.
  - `src/pages/`: Application pages (Dashboard, Home, Login, etc.).
  - `src/components/`: Reusable components (Navbar, Footer, etc.).
